## Concept

Rebuild the Canva idea as a proper short-form ad sequence:

1. Intro hook
2. Mistake 1
3. Mistake 2
4. Mistake 3
5. Mistake 4
6. CTA close

## Design corrections vs Canva draft

- One message per frame instead of stacking everything into one screen.
- Stronger vertical rhythm and clearer safe margins.
- Consistent beige headline block + charcoal detail block system.
- Cleaner number badges and stronger reading order.
- Shortened body text for video consumption.

## Suggested edit/export workflow

1. Import the six PNG files into Canva.
2. Put each PNG on its own page.
3. Set durations close to the GIF preview timing.
4. Add gentle crossfades or no transition at all.
5. Export as MP4 from Canva.

## Note

The environment here does not include `ffmpeg`, so a true MP4 was not rendered locally. A GIF preview is included for motion-flow review.
